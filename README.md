# Aether One Landing Page

A modern Apple-inspired static landing page with:
- Storytelling hero layout
- Fluid reveal animations and interactive tilt device card
- Responsive design with accessibility-conscious motion handling
- Basic SEO + social metadata

## Local preview

```bash
python3 -m http.server 8080
# open http://localhost:8080
```

## Deploy
